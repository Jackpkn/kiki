# Kiki's Travelogue ✈️

A modern travel companion website migrated to **Next.js 15** with App Router.

## Features

- **Next.js Backend**: Uses Next.js API routes for fetching destinations.
- **App Router**: Modern React patterns with Server Components and Client Components.
- **Beautiful UI**: Enhanced with Framer Motion, Lucide icons, and Tailwind CSS.
- **Interactive**: Filter destinations, search, and add new places.
- **Dark Mode**: Smooth transition between light and dark themes.

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Components**: Shadcn/UI
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **State Management**: TanStack Query & React Hooks

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app`: Next.js App Router (Pages, Layouts, API Routes)
- `src/components`: Reusable UI components
- `src/data`: Static data and mock destinations
- `src/types`: TypeScript definitions
